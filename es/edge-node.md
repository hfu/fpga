# Edge Node

---

## Definición Conceptual

Un edge node es una posición en un sistema donde se toman decisiones bajo restricciones.

No un héroe. No una plataforma. No un estado. Un lugar específico donde convergen autonomía, responsabilidad y consecuencias.

---

## Características

Los edge nodes operan donde:

- **La conectividad es intermitente** — La red está disponible a veces, no siempre
- **La coordinación central no está disponible** — Las decisiones no pueden esperar aprobación
- **Se requiere toma de decisiones local** — La acción ocurre en el borde, no en el centro

Los edge nodes no fallan ante la desconexión. La desconexión es la condición operativa normal.

---

## Escenario

Un ingeniero de campo soluciona problemas de infraestructura energética en un lugar remoto.

**Contexto:**
- Sin señal de teléfono móvil
- Energía de batería limitada a 6 horas
- Los datos de diagnóstico se almacenan localmente
- Las decisiones de reparación deben tomarse ahora

**Lo que sucede:**

El ingeniero usa una tableta con herramientas de diagnóstico offline. El sistema incluye:
- Manuales de equipos
- Datos históricos de fallos
- Árboles de decisión para problemas comunes
- Almacenamiento de datos local

El ingeniero identifica el fallo, realiza la reparación y registra el resultado. Cuando vuelve la conectividad, los datos se sincronizan con la plataforma central. La plataforma agrega patrones a través de miles de reparaciones. Pero la plataforma no tomó la decisión. El edge node lo hizo.

**Por qué esto importa:**

Si el sistema requiere acceso a la nube para funcionar, la reparación no ocurre. El edge node está bloqueado por arquitectura, no por capacidad. Esto es falla de diseño, no falla de campo.

---

## El Cambio

Marco antiguo: Estado versus individuo.

Nuevo marco: Plataforma y campo.

El campo siempre ha existido. Clínicas remotas. Trabajo offshore. Respuesta a desastres. Estaciones de investigación. El cambio no es que el campo se vuelva real. Es diseñar sistemas que reconozcan la realidad del campo como punto de partida, no como caso extremo.
